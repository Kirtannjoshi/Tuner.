# Tuner 📻

Tuner is a premium, open-source global radio streaming indexer built on top of the Radio-Browser API. 
Combining a state-of-the-art Web Audio API processing engine with buttery-smooth React aesthetics, Tuner bridges the gap between massive global radio networks (Nano Services) and pristine architectural design.

## Features

- **Global Smart Search**: Utilizing debounced micro-requests, search through over 40,000 live streams instantly.
- **Hi-Fi Audio Processor**: An integrated Web Audio API tree with Dynamics Compression and High/Low shelf EQ routing mimics Dolby/B&O studio-grade crispness.
- **Premium Nano Services**: Select heavy-hitting broadcasting groups (like BBC Radio or Capital FM) seamlessly.
- **Liquid Interface**: Fully driven by `framer-motion` for spring-based physics, layout animations, and fluid tab transitions.

## Architecture & Disclaimer
Tuner does **not** host, own, or manage any broadcasting content. It acts as an open indexer wrapper mapping to public `.m3u8` and `.mp3` streams via public directory indexes. There are **zero** payment gateways or paywalls integrated.

## Getting Started

1. `cd web`
2. `npm install`
3. `npm run dev`

Navigate to `http://localhost:5173` to experience the crispest radio UI on the web.
